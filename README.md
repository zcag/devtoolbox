# DevToolBox

**[devtoolbox.dev](https://zcag.github.io/devtoolbox/)** — 25 free online developer tools that run entirely in your browser. No data ever sent to a server.

## Tools

| Tool | Description |
|------|-------------|
| [JSON Formatter](https://zcag.github.io/devtoolbox/json/) | Format, validate, and minify JSON |
| [Base64 Encoder/Decoder](https://zcag.github.io/devtoolbox/base64/) | Encode and decode Base64 strings |
| [JWT Decoder](https://zcag.github.io/devtoolbox/jwt/) | Decode and inspect JSON Web Tokens |
| [Hash Generator](https://zcag.github.io/devtoolbox/hash/) | Generate MD5, SHA-1, SHA-256, SHA-512 hashes |
| [URL Encoder/Decoder](https://zcag.github.io/devtoolbox/url/) | Encode and decode URL components |
| [Timestamp Converter](https://zcag.github.io/devtoolbox/timestamp/) | Convert between Unix timestamps and dates |
| [UUID Generator](https://zcag.github.io/devtoolbox/uuid/) | Generate v4 UUIDs |
| [Text Diff](https://zcag.github.io/devtoolbox/diff/) | Compare two texts side by side |
| [Color Converter](https://zcag.github.io/devtoolbox/color/) | Convert between HEX, RGB, HSL |
| [Regex Tester](https://zcag.github.io/devtoolbox/regex/) | Test regular expressions with live matching |
| [Cron Parser](https://zcag.github.io/devtoolbox/cron/) | Parse and validate cron expressions |
| [YAML/JSON Converter](https://zcag.github.io/devtoolbox/yaml/) | Convert between YAML and JSON |
| [Markdown Preview](https://zcag.github.io/devtoolbox/markdown/) | Live markdown rendering |
| [Password Generator](https://zcag.github.io/devtoolbox/password/) | Generate secure passwords |

| [Lorem Ipsum Generator](https://zcag.github.io/devtoolbox/lorem/) | Generate placeholder text |
| [Chmod Calculator](https://zcag.github.io/devtoolbox/chmod/) | Calculate Unix file permissions |
| [HTML Entity Encoder](https://zcag.github.io/devtoolbox/html-entity/) | Encode/decode HTML entities |
| [Text Case Converter](https://zcag.github.io/devtoolbox/case/) | Convert text between cases |

| [CSS Box Shadow Generator](https://zcag.github.io/devtoolbox/box-shadow/) | Create CSS box shadows with live preview |

## Why DevToolBox?

- **100% client-side** — your data never leaves the browser
- **No signup, no ads** — just tools
- **Fast** — static site, no server round-trips
- **Open source** — contribute or fork
